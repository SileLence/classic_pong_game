# Classic Ping Pong Game

This is a modern reinterpretation of the classic Pong mechanics, built with Java and LibGDX.
The project focuses on precise physics, smooth controls, polished visual style, and responsive gameplay for both single-player and two-player modes.

This is an active development project with continuous improvements in visuals, physics, and UI/UX.

## Author
